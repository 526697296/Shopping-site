### 实现的功能
1. 音乐播放、暂停、上一曲、下一曲
2. 播放列表、添加到播放列表、做为下一首播放
3. 搜索单曲、歌手、专辑、MV
4. 查看歌手页面、专辑页面、MV
5. 热门搜索
6. 搜索历史记录
7. 排行榜
8. 切换播放模式
9. 歌词

### 未实现的功能
*   我喜欢
*   下载

## Build Setup

``` bash
# 下载或克隆下来，然后安装依赖
npm install

# 开发预览
npm run dev

# 打包发布，生成的文件在dist文件夹中
npm run build
```
